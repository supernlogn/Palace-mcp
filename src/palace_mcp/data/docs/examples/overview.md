Some examples of using *Palace*, including configuration and mesh files, can be found in the [`examples/`](https://github.com/awslabs/palace/blob/v0.16.0/examples) directory of the source code. The following sections provide complete tutorials for each of the available example applications.

These examples are also used by *Palace*'s regression testing suite. See the [`test/examples/`](https://github.com/awslabs/palace/blob/v0.16.0/test/examples/) directory for more details.
